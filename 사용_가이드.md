# 📊 Reach 중복 제거 계산기 - 실행 가이드

## 🚀 빠른 시작

### 1단계: Python 설치 확인
```bash
# 터미널(명령 프롬프트)에서 실행
python --version
```
Python 3.8 이상이 필요합니다. 없다면 https://www.python.org/downloads/ 에서 다운로드하세요.

### 2단계: 필요한 패키지 설치
```bash
# 터미널에서 실행 (파일이 있는 폴더로 이동 후)
pip install -r requirements.txt
```

### 3단계: 앱 실행
```bash
# 터미널에서 실행
streamlit run reach_calculator_app.py
```

자동으로 웹 브라우저가 열리며 앱이 실행됩니다! 🎉

---

## 📖 상세 사용 방법

### 💡 방법 1: 직접 입력

1. **채널 추가**
   - "새 채널 이름" 입력란에 채널명 입력 (예: MBC, EBS, CATV)
   - "➕ 채널 추가" 버튼 클릭

2. **소재 추가**
   - 각 채널 섹션에서 "소재 이름" 입력 (예: 버스_15s)
   - "➕ 소재 추가" 버튼 클릭

3. **Reach 데이터 입력**
   - 각 소재별로 Reach 1+, 2+, 3+ 값 입력

4. **계산**
   - 모든 데이터 입력 후 "🚀 계산 실행" 버튼 클릭

### 📁 방법 2: CSV 파일 업로드

#### 옵션 A: 통합 CSV 파일
CSV 파일을 다음 형식으로 준비하세요:

| Channel | Creative | Reach 1+ | Reach 2+ | Reach 3+ |
|---------|----------|----------|----------|----------|
| MBC     | 버스_15s  | 45936    | 9586     | 4378     |
| MBC     | 회사_15s  | 45046    | 9808     | 4412     |
| EBS     | 버스_15s  | 8411     | 2106     | 1046     |

#### 옵션 B: 채널별 개별 CSV 파일
파일명 형식: `채널명_소재명.csv` (예: mbc_버스.csv)

파일 내용 (마지막 행 사용):
```
날짜,Reach 1+,Reach 2+,Reach 3+
2025-11-01,45936,9586,4378
2025-11-02,46000,9600,4400
```

### ⚙️ 중복률 조정

사이드바에서 중복률을 조정할 수 있습니다:
- **Reach 1+ 중복률**: 기본 35% (실증 데이터: 32%)
- **Reach 2+ 중복률**: 기본 25%
- **Reach 3+ 중복률**: 기본 15%

### 🎯 Grand Total 입력 (선택사항)

Grand Total을 알고 있다면:
1. 사이드바에서 "Grand Total 알고 있음" 체크
2. Reach 1+, 2+, 3+ 값 입력
3. 방법 2 (비례 조정) 결과 확인 가능

## 🔧 문제 해결

### Q: 앱이 실행되지 않아요
**A**: 다음을 확인하세요:
```bash
# Python 버전 확인
python --version

# 패키지 재설치
pip install -r requirements.txt --upgrade

# Streamlit 재설치
pip uninstall streamlit
pip install streamlit
```

### Q: CSV 업로드 오류가 나요
**A**: CSV 파일 형식을 확인하세요:
- UTF-8 인코딩 사용
- 컬럼명 정확히 입력
- 숫자 데이터에 쉼표(,) 없이 입력

### Q: 브라우저가 자동으로 열리지 않아요
**A**: 터미널에 표시된 주소를 수동으로 입력하세요:
```
http://localhost:8501
```

### Q: 계산 결과가 이상해요
**A**: 다음을 확인하세요:
- 모든 Reach 값이 올바르게 입력되었는지
- Reach 2+ ≤ Reach 1+ 인지
- Reach 3+ ≤ Reach 2+ 인지

## 📱 모바일에서 사용하기

Streamlit 앱은 모바일 브라우저에서도 작동합니다!

1. **같은 네트워크에서**:
   - PC에서 앱 실행
   - 터미널에서 Network URL 확인
   - 모바일 브라우저에서 해당 URL 접속

2. **Streamlit Cloud에 배포** (무료):
   - GitHub에 코드 업로드
   - https://streamlit.io/cloud 에서 배포
   - 언제 어디서나 접속 가능!

## 🌐 인터넷에 배포하기 (무료)

### Streamlit Cloud 배포 (추천)

1. **GitHub 계정 만들기**
   - https://github.com 에서 가입

2. **저장소(Repository) 만들기**
   - "New repository" 클릭

   - `requirements.txt`

4. **Streamlit Cloud에 배포**

   - "New app" 클릭
   - GitHub 저장소 선택
   - 배포 완료!

5. **공유하기**
   - 생성된 URL을 동료들과 공유
   - 예: `https://your-app.streamlit.app`
